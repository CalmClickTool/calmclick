CalmClick — local / offline copy
================================

You downloaded the private-friendly version. No install wizard. No account.

HOW TO OPEN
-----------
1. Unzip this folder anywhere (Desktop is fine).
2. Open the folder.
3. Double-click  index.html
4. It opens in your browser. You’re done.

TIP: Right-click index.html → Open with → Chrome or Edge if double-click
uses a browser you don’t like.

WHAT YOU GET
------------
- Check a link
- Check a message (scam patterns)
- Explain a computer error
- “How do I…?” everyday guides

PRIVACY
-------
Everything runs on your computer. What you paste is not uploaded.

OPTIONAL: BROWSER EXTENSION
---------------------------
Inside the “extension” folder (if included) you can load CalmClick into
Chrome or Edge:

1. Open chrome://extensions  (or edge://extensions)
2. Turn on Developer mode
3. Load unpacked → select the “extension” folder
