How to install the extension (Chrome or Edge)

Open in Chrome Web Store

Click Add to Chrome (or Edge). That’s it.

  1. Download the extension zip and unzip it to a folder you’ll keep (don’t delete the folder later).
  2. Open chrome://extensions (Chrome) or edge://extensions (Edge).
  3. Turn on Developer mode (top right).
  4. Click Load unpacked and select the unzipped extension folder.
  5. Pin CalmClick from the puzzle-piece menu if you like. Right‑click any link → Check with CalmClick.

When the Chrome Web Store listing is live, this page will point you there for one‑click install. Sideloading stays available forever for the local / offline crowd.
